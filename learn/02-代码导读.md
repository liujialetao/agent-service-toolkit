# 代码导读

## 目录结构

```
agent-service-toolkit/
├── .env                      # 本地密钥与配置（勿提交）
├── .streamlit/
│   └── credentials.toml      # 跳过 Streamlit 首次邮箱提示
├── .venv/                    # uv 虚拟环境
├── learn/                    # 本目录：学习与启动笔记
├── src/
│   ├── streamlit_app.py      # ★ 前端入口（Streamlit UI）
│   ├── run_service.py        # ★ 后端启动入口
│   ├── client/
│   │   └── client.py         # HTTP 客户端，调用 FastAPI
│   ├── service/
│   │   └── service.py        # FastAPI 路由（/info、/invoke、/stream）
│   ├── agents/               # ★ LangGraph Agent 定义（学习重点）
│   │   ├── chatbot.py
│   │   ├── research_assistant.py
│   │   ├── rag_assistant.py
│   │   └── agents.py         # Agent 注册表
│   ├── core/
│   │   ├── settings.py       # 环境变量、模型配置
│   │   └── llm.py            # LLM 工厂（含 openai-compatible）
│   ├── schema/               # 请求/响应数据结构
│   └── voice/                # 语音输入输出（可选）
└── docs/                     # 官方文档（RAG、Ollama 等）
```

---

## 前后端分工

| 层 | 文件 | 职责 |
|----|------|------|
| 前端 | `streamlit_app.py` | 聊天界面、侧边栏、消息渲染、Tool Call 展示 |
| 客户端 | `client/client.py` | 封装对后端的 HTTP 调用 |
| 后端 | `service/service.py` | REST API、流式 SSE、错误处理 |
| Agent | `agents/*.py` | LangGraph 图定义、工具、Prompt |
| 配置 | `core/settings.py` | 读取 `.env`，决定可用模型 |

前端连接后端的关键代码（`streamlit_app.py`）：

```python
agent_url = os.getenv("AGENT_URL") or f"http://{host}:{port}"
st.session_state.agent_client = AgentClient(base_url=agent_url)
```

---

## `streamlit_app.py` 主要函数

| 函数 | 作用 |
|------|------|
| `main()` | 页面布局、连接后端、侧边栏 Settings |
| `draw_messages()` | 渲染聊天记录、流式输出、Tool Call |
| `handle_feedback()` | 五星反馈 |
| `handle_sub_agent_msgs()` | Supervisor 多 Agent 消息 |

侧边栏可选择 **Agent** 和 **LLM Model**，对应 Settings 弹窗。

## Agent 学习顺序（建议）

| 顺序 | Agent key | 说明 | 阅读文件 |
|------|-----------|------|----------|
| 1 | `chatbot` | 最简单聊天，无工具 | `src/agents/chatbot.py` |
| 2 | `research-assistant` | Web 搜索 + 计算器 | `src/agents/research_assistant.py` |
| 3 | `rag-assistant` | RAG 问答 | `src/agents/rag_assistant.py`、`docs/RAG_Assistant.md` |
| 4 | `langgraph-supervisor-agent` | 多 Agent 协作 | `src/agents/langgraph_supervisor_agent.py` |

完整列表：

```bash
curl -s http://127.0.0.1:8080/info | python3 -m json.tool
```

## 与 LangChain-RAG-FastAPI-Service 对比

| | agent-service-toolkit | 原 RAG 笔记项目 |
|--|----------------------|----------------|
| Agent 编排 | LangGraph（StateGraph） | LangChain AgentExecutor |

| RAG | `rag-assistant` + ChromaDB | 知识库 + HyDE + 重排序 |
